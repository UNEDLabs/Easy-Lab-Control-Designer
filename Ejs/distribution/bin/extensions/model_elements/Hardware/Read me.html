Hardware model elements

The Hardware directory groups model elements devoted to help you connect to equipment connected to your computer via the USB port.

The elements in this group are:

  • Arduino encapsulates access to a Arduino board.
  • CoachLab encapsulates access to a CoachLab II+ interface, sold by CMA.
  • The different Phidget elements provide access to a Phidget cards, sold by Phidgets Inc..

More info about model elements in the Easy Java Simulations' Wiki.
