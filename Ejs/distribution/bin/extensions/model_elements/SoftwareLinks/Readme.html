SoftwareLinks model elements

The SoftwareLinks Library contains elements that can be used to easily link EJS with external apps like LabVIEW, MATLAB, etc.

The elements available in this library are:

  • LabVIEW v1.0 Element. This element connects to a deprecated JIL server.
  • LabVIEW v2.0 Element. This element connects to a JIL server using the XML-RPC protocol. The server can be found and downloaded at https://github.com/UNEDLabs/jil-server
  • RIP-MATLAB Element. This element connects to Matlab programs through a JIM server using the RIP protocol. The server can be found and downloaded at https://github.com/UNEDLabs/jim-server
  • RIP-Simulink Element. This element connects to Simulink models through a JIM server using the RIP protocol. The server can be found and downloaded at https://github.com/UNEDLabs/jim-server
  • RIP-Node.js Element. This element connects to a Node.js server using the RIP protocol (based on JSON-RPC and Websockets). The server can be found and downloaded at https://github.com/UNEDLabs/rip-js-server

Up-to-date versions of the elements can be found at Github.
