Numerics model elements

The Numerics directory groups model elements that provide access to third-party numerical libraries.

The elements in this group are:

  • NumericJS provides access to the Numeric Javascript library created by Sébastien Loisel.
  • MathJax is an open-source JavaScript display engine for LaTeX, MathML, and AsciiMath notation available at MathJax GitHub.
  • Math.js is an extensive math library for JavaScript available at MathJS website.

More info about model elements in the Easy Java Simulations' Wiki.
