Using your own JAR libraries

This example shows how to use user defined JAR files in  EJS. The JAR file used here is quite simple. It just contains a class with a single method to compute the thrid power of a double number.

The steps to follow are:

  1. Create your source code which must be in one or more packages. (See the sample code for constructing a function f(x) = x*x*x in the description pages of this model.)
  2. Compile this code properly.
  3. Create a JAR file (using Java's jar utility) for the generated classes.
  4. Copy this JAR file into your workspace source directory (preferably under the same directory of your source XML file).
  5. Open the simulation information panel (top-right icon of EJS) and add this JAR file to the list of 'JAR Libraries' as indicated in the 'Option and files' tab of the panel.

You can now use this method (fully qualified) as any other Java method. In our example, the method is called as in: mypackage.MyMathClass.power3(x);

If you want to make it easy to use this method, you can add mypackage.MyMathClass to the list of import statements of the 'Option and files' tab of the simulation information panel, and then call the method simply as in : MyMathClass.power3(x);
