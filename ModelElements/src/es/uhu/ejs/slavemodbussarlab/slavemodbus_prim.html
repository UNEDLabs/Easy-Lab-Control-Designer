Slave Modbus Element

Other available methods.

From now on the most representative methods of the element are presented. Other advanced methods are included in an anexed page accesible by means of the icon "Advanced methods".

  • short readAnalogInput(int ind); //This function reads the value from an analog input pin.
  • short[] readAnalogInputs(int ind, int len); //This function reads the value from several analog input pins.
  • short readAnalogRegister(int ind); //This function reads an analog value from a register.
  • short[] readAnalogRegisters(int ind, int len); //This function reads a list of analog value from registers and returns it as a short array.
  • boolean readCoil(int ind); //This function reads the value from the input digital pin whose position is passed as parameter.
  • boolean[] readCoils(int ind, int len); //This function reads the values from some input digital pins.
  • boolean readDigitalInput(int ind); //This function reads the value from the input digital pin whoe position is passed as parameter.
  • boolean[] readDigitalInputs(int ind, int len); //This function reads the values from a list of the input digital pins.
  • void writeAnalogInput(int ind, short value); //This function writes a short value to an analog output pin.
  • void writeAnalogInputs(int ind, short[] value); //This function writes an array of short values to several analog output pins.
  • void writeAnalogRegister(int ind, short value); //This function writes a short value to an analog register.
  • void writeAnalogRegisters(int ind, short[] value); //This function writes a list of short values to some analog registers.
  • void writeCoil(int ind, boolean value); //This function writes the value passed as parameter to the output digital pin whose position is also passed as parameter.
  • void writeCoils(int ind, boolean[] value); //This function writes the values passed as parameters to a list of output digital pins.
  • void writeDigitalInput(int ind, boolean value); //This function writes the value passed as parameter to the output digital pin whose position is also passed as parameter.
  • void writeDigitalInputs(int ind, boolean[] value); //This function writes the values passed as parameters to some output digital pins.
