Slave Modbus Element

This is an EJS element that allows you to maintain an image of the values contained in the input / output pins of a data acquisition card. You can update the values of the output pins received from the master to the image and the values of the image of the input pins with the received from the card.

Usage

To add a Slave Modbus Element to your model, simply drag the icon element to the list of your model elements. You must set the following properties:

  • IP address of the server.
  • Port used by the server.
  • idSlave  of the identifier slave.

Available methods

To use this element an only method is really needed:

  • void run(); //This function updates the input pins with the internal structure.

Once executed the method run, you can access the image of the process by means of the next variables:

  • boolean[] C = new boolean[numberCoilOutputs]; //Variable to access the digital output pins of the board.
  • boolean[] I = new boolean[numberDigitalInputs]; //Variable to access the digital input pins of the board.
  • short [] IR = new short[numberAnalogInputs]; //Variable to access the analog input pins of the board.
  • short [] R = new short[numberAnalogRegister]; //Variable to access the analog output pins of the board.
