Master Modbus Element

This EJS element allows you to send values to the output pins of a board and receive the values from its input pins by means of an image of the board structure.

Usage

To add a Master Modbus Element to your model, simply drag the icon element to the list of your model elements. You must set the following properties:

  • IP address of the server.
  • Port used by the server.
  • idSlave of the identifier slave.

Available methods

From now on the most representative methods of the element are presented. Other advanced methods are included in an anexed page accesible by means of the icon "Advanced methods"
