Master Modbus Element. Advanced Methods

Advanced methods to make complex operations

  • int getRetries(); //This function returns the connection number of retries.
  • int getSlaveId(); //This function returns the slave id.
  • int getTimeout(); //This function returns the value of the parameter timeout.
  • int getRetries(); //This function returns the connection number of retries.
  • double readAnalogInputToDouble(int ind); // This function reads the value from the image of an analog input as double.
  • double readAnalogInputToFloat(int ind); // This function reads the value from the image of an analog input as float.
  • int readAnalogInputToInteger(int ind); // This function reads the value from the image of an analog input as integer.
  • long readAnalogInputToLong(int ind); // This function reads the value from the image of an analog input as long.
  • int readAnalogInputToShortUnsigned(int ind); // This function reads the value from the image of an analog input as unsigned short.
  • double readAnalogRegisterToDouble(int ind); // This function reads the value from an analog input of the image as double.
  • double readAnalogRegisterToFloat(int ind); // This function reads the value from an analog input of the image as float.
  • int readAnalogRegisterToInteger(int ind); // This function reads the value from an analog input of the image as int.
  • long readAnalogRegisterToLong(int ind); // This function reads the value from an analog input of the image as long.
  • int readAnalogRegisterToShortUnsigned(int ind); // This function reads the value from an analog input of the image as unsigned short.
  • int readAnalogRegisterToShortUnsigned(int slaveId, int ind); // This function reads the value from an analog input of the image as unsigned short.
  • byte readExceptionStatus(int slaveId); //This function read the status of the exceptions of the slave.
  • byte[] reportSlaveId(int slaveId); //This function reads the Id of slave connected to the board and return it as an array of bytes.
  • void setRetries(int retries); //This function sets the default number of retries of MasterModbus class to the passed value.
  • void setSlaveId(int slaveId); //This function sets the slave id to the passed value.
  • void setTimeout(int timeout); //This function sets the default parameter Timeout of MasterModbus class to the passed value.
  • boolean writeAnalogRegisterDouble(int ind, double value); //This function writes a double value to the image of the analog output of Arduino.
  • boolean writeAnalogRegisterFloat(int ind, float value); //This function writes a float value to the image of the analog output of Arduino.
  • boolean writeAnalogRegisterInteger(int ind, int value); //This function writes an int value to the image of the analog output of Arduino.
  • boolean writeAnalogRegisterLong(int ind, long value); //This function writes a long value to the image of the analog output of Arduino.
  • boolean writeAnalogRegisterShort(int ind, short value); //This function writes a short value to the image of the analog output of Arduino.
  • boolean writeAnalogRegisterShortUnsigned(int ind, int value); //This function writes an unsigned short value to the image of the analog output of Arduino.
  • void writeMaskRegister(int slaveId, int offset, int and, int or); //This function writes the value of the image of an analog outputs of Arduino using a mask.
