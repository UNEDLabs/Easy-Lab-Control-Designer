Master Modbus Element

This EJS element allows you to send values to the output pins of a board and receive the values from its input pins by means of an image of the board structure.

Usage

To add a Master Modbus Element to your model, simply drag the icon element to the list of your model elements. You must set the following properties:

  • IP address of the server.
  • Port used by the server.
  • idSlave of the identifier slave.

Available methods

From now on the most representative methods of the element are presented. Other advanced methods are included in an anexed page accesible by means of the icon "Advanced methods"

  • boolean connect();//This function connects the MasterModbus class and the board using default values of necessary parameters.
  • void close();//This function disconnects the MasterModbus class and the board.
  • void monitorInit(); //This function creates a monitor to make easy the test of all the functions included in the MasterModbus class.
  • short readAnalogInput(int ind); //This function reads the value from the image of an analog input.
  • short[] readAnalogInputs(int start, int len); // This function reads the values from the image of several analog inputs as short.
  • short readAnalogRegister(int ind); // This function reads the value from an analog input of the image as short.
  • short[] readAnalogRegisters(int start, int len); // This function reads the values from several analog inputs of the image as short.
  • boolean readCoil(int ind); // This function reads the value from a digital input of the image.
  • boolean[] readCoils(int start, int len); // This function reads the values from several digital inputs of the image.
  • boolean readDigitalInput(int ind); //This function reads the value from a digital input of the image.
  • boolean[] readDigitalInputs(int start, int len); //This function reads the values from several digital inputs of the image.
  • boolean writeAnalogRegister(int offset, int value); //This function writes the value from an analog output to the image.
  • boolean writeAnalogRegisters(int start, short[] values); //This function writes the values to the image of several analog outputs.
  • boolean writeCoil(int ind, boolean value); //This function writes a value in a digital output of Arduino to the image.
