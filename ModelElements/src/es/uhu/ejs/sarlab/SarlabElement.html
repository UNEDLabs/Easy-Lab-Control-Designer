Sarlab element

SARLAB element provides remote connectivity to laboratory experiments that are managed by a SARLAB server.

Usage

         To add a SARLAB element to your model, simply drag the icon element to the list of your model elements. You must set the following properties:

  • IP address of the server.
  • Port used by the server.
  • SARLAB lab experience identifier.
  • Username and password. Optional. If not specified, the server will submit a popup login window.

If you pack the EJS application with Moodle support, the element will attempt to read the parameters from the applet.

Available methods

The element provides six methods:

  • public void connect(); Establishs a connection to an experience using the SARLAB server. Reads the parameters from the element properties or from the applet.
  • public void connect(String ipServer,String portServer,String idExp).
  • public void connect(String ipServer,String portServer,String user,String password,String idExp).
  • public void disconnect();  Disconnect from SARLAB server.
  • reloadconnect(): Recharges the connection.
  • public boolean openconnect(); Returns true if the connection is established, false otherwise.

Examples of use

sarlab.connect();

sarlab.connect("192.168.1.3", "443", "VHDL Lab #1");
