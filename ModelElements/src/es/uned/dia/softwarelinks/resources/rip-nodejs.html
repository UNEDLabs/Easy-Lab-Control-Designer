RIP-Node.js model element

A RIP model element provides an implementation of the Remote Interoperability Protocol (RIP) API.


Usage

To add a RIP model element to your model, drag the icon element to the list of your model elements and set the name of the variable for referring to the element. The connection can be configured in the preferences window, the address of the server, and the port to establish the connection. The links between the EJS variables and the server variables also can be done in this window.

The main methods of the API are:

  • connect(): Creates a new connection with a RIP Server.

  • open(): Opens a resource.

  • get(): Reads one or more variables.

  • set(): Writes one or more variables.

  • step(): Writes one or more variables, execute a command and read one or more variables.

  • close(): Closes an opened resource.

  • disconnect(): Disconnects from server.

Example of use

The following code is usually included into an initialization page, and it must be executed to set up the connection (we use ripclient as instance name of the element) 

        ripclient.connect();

The following code is usually included into an evolution page, to exchange data periodically with the server. 

        ripclient.step();

The following code must be executed to finalize the execution properly. 

        ripclient.disconnect();
