Audio Player model element

The Audio Player element allows to play audio streaming. The audio formats currently supported are G.711 (mu-law, a-law), and AAC.

Usage

To add an audio Player element to your model, drag one block icon element to the list of your model elements. The element provide the following methods,

  • start(), to initiate the audio
  • stop(), to pause it.
  • setUrl(), to define the audio streaming source
