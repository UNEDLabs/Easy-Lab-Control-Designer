Socket Comm model element

A socket comm element allows establishing a socket communication mechanisms between applications.

Usage

To add a SocketComm reader element to your model, drag the icon element to the list of your model elements and edit the "Input" and "Output" editors to provide the communication protocol.

The "Input" code is run autommatically whenever data is received through the socket. The "Output" code must be explicitily invoked through the sendData() method.

The element implements the following basic methods:

  • boolean start(): Starts the communication
  • boolean stops(): Stops the communication
  • boolean sendData(): Runs the "Output" code to prepare the output data and the sends it the streaming data.

Example of use

socketComm.start(); // Starts the connection 
socketComm.sendData(); // Sends the output data
socketComm.stop(); // Stops the connection
