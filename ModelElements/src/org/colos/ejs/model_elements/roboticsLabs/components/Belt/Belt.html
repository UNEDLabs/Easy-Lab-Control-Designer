Belt element

This element adds a belt element to the robotics lab. This element represents a conveyor belt used to carry objects from one point to another. The object are moved, over a horizontal plane, in continuous motion.

Usage

The belt can be turned on and off using the connect() and disconnect() methods, respectively.

The author can also edit a customization dialog for the belt. Typically, Length, Width and High customization fields must be filled to change the dimensions of the belt.

As with the robots, the author also chooses a 3D group View element for display.

Other methods available for this element are:

    - Information about the belt:

  • public double getLength(): Gets the length of the belt
  • public double getWidth(): Gets the width of the belt
  • public double getHigh(): Gets the high of the belt

    • - Configuration of the belt:

  • public double getLinealDistance(): Gets the linear distance of the belt in meters/seconds
  • public double getNumberOfPulses(): Gets the number of encoder pulses
  • public int getVariatorFrequency(): Gets the actual frequency of the belt variator
  • public void setVariatorFrequency(int variatorFrequency): Sets the variator frequency of the belt (unit in Hz)

    • - Configuration of the robotics laboratory:

  • public void attachObject(Element object): Puts the object in the belt
  • public void detachObject(Element object): Removes the object of the belt
