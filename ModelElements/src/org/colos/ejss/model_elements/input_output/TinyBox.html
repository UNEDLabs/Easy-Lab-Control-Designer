NumericJavascript model element

TinyBox encapsulates access to the TinyBox2 package.

Usage

To add the TinyBox element to your model, drag the icon element to the list of your model elements.

The element declares a new global variable (object) called TINY.box which should be treated a reserved word in your simulation. You can use the function show of this new object either as function of the new object TINY.box, or of the object named as your element (which is just an alias for TINY.box).
