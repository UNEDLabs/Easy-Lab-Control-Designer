MathJax model element

MathJax is an open-source JavaScript display engine for LaTeX, MathML, and AsciiMath notation available at MathJax GitHub.

Usage

To add the MathJax element to your model, drag the icon element to the list of your model elements.

The element declares a new global variable (object) called MathJax which should be treated a reserved word in your simulation.

Read the documentation for details.

Internal configuration for default

Ejs configures MathJAX using the following parameters:

  MathJax.Hub.Config({
	jax: ["input/TeX", "output/SVG"],
	extensions: ["tex2jax.js", "fast-preview.js"],
	TeX: {
		extensions: ["AMSmath.js","AMSsymbols.js",
				 "noErrors.js","noUndefined.js"]
	}});

Refresing display

    	MathJax.Hub.Queue(["Typeset",MathJax.Hub]);
